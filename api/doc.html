forkort.no

Slå opp en ID (get.php)

API for oppslag av forkortede URL-er kan gjøres via HTTP GET- eller HTTP POST-forespørsler. Forventer kun ett parameter, 'id', ID-en som skal slås opp.

Legg inn ny URL (add.php)

API-et aksepterer POST-requests med ett parameter, 'url', URL-en som skal forkortes.

Dra denne lenken forkort til bokmerkene dine for raskere tilgang.
Vi forbeholder oss retten til å slette lenker som kan virke støtende og til å stenge ute IP-adresser som misbruker denne tjenesten.
